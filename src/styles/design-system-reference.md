## Populated Main Page Layout
- Page background: `--semantic-color-surface-page`
- Desktop page padding: `--spacing-1100` top/bottom and `--spacing-500` left/right
- Tablet page padding: `--spacing-500`
- Mobile page padding: `--spacing-300`
- Main desktop content width: `--size-layout-content-max-width`
- Desktop column split: `800px` product list and `384px` cart with `--spacing-400` gap
- Product grid gap: `--spacing-300` columns, `--spacing-400` rows on tablet/desktop, `--spacing-300` rows on mobile

## Populated Cart Panel
- Card surface: `--semantic-color-surface-raised`
- Card radius: `--spacing-150`
- Card internal spacing: `--spacing-300`
- Cart title color: `--semantic-color-text-accent`
- Row separators: `--semantic-color-border-subtle`
- Quantity accent color: `--semantic-color-text-accent`
- Carbon info background: `--semantic-color-surface-page`
